### Run:
```bash
npm install
gulp / gulp build / gulp watch / gulp serve
```

### Testing:
```bash
npm run test
npm run cypress
```

Firefox has to be installed as it is used in headless mode.

### Git hooks:
Execute install.sh from githooks folder
